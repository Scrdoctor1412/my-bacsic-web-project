ReCaptcha error when submitting a support ticket via the webform

March 2 2022 7:20am

If you are a receiving a ReCaptcha error when attempting to submit a support ticket via the support webform, this is likely caused by adblocking or js-blocking browser extensions or software. Please add an exception for the Framework website in the configuration of those extensions or software in order to successfully receive the ReCaptcha check and submit your support ticket.
